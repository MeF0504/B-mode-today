# B mode today!

Plot CMB B-mode power spectrum and data points.

## How to make plot
```shell
# show usage
python3 main.py -h
```
``` shell
# run by default settings
python3 main.py
```

## Requirements
- [python3](https://www.python.org/downloads/)
- [numpy](https://numpy.org/)
- [camb](https://camb.readthedocs.io/)
- [matplotlib](https://matplotlib.org/)
